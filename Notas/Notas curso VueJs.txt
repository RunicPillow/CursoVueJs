- API Styles
Los componentes de Vue.js se pueden escribir en dos APIs diferentes:

Options API y Composition API

- State - determina la situación actual de nuestra aplicación
Vue.js tiene un par de funciones: Reactive y Ref que nos permitiran declararlo y sobretodo identificar cuando el state ha cambiado.
Solo estan disponibles en Composition API

- Directivas - Atributos HTML con JavaScript, inician con v- y se colocan como atributos HTML


- Props - sirven para comunicar componentes, los props pueden ser datos estaticos o reactivos.
Los props nunca deben modificar el state en el componente hijo


- Eventos
tipos de eventos: 
Inline Handlers - son para tareas muy sencillas como cambiar a dark o light mode
Method Handlers - son muy utiles cuando realizaras diferentes acciones como validar los datos de un formulario

v-on:click="incrementar"
@click="incrementar"
